Overview
========
Java Computer Vision Library (`JcvLib`) is a Computer Vision and Image Processing Library for Java.


Requirements
============
 * Java 1.8 or newer.
 * If you want using video -- *OS*: Linux or Windows, *Arch*: i386 or x86/x64.
